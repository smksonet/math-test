1. What is DOM and what is the purpose of DOM?

    2. Ans: DOM means Document Object Model.It represents our page in such a way that it can be easily modified with programming languages. Creating a new element,Remove any previous element,Modify any previous element,Modify, add, remove attributes,Add, remove, modify styling,Create events on the page to perform a specific action on the page ,when something like this happens,Creating events on a specific element, when something happens on this element we want to take such an action.

  2. How will you select HTML elements using DOM? Name the DOM methods.

    3. Ans:The easiest way to access a single element in the DOM is by its unique ID.Logging demo Id to the console will return our entire HTML element.There are five ways in which we can select elements in a DOM using selectors.
    • getElementsByTagName()
    • getElementsByClassName()
    • querySelectorAll()
    • getElementById()
    • querySelector()

  3. What is event bubble?

    4. Ans: If there are two elements element 1 and element 2. Element 2 is inside element 1 and we attach an event handler with both the elements lets say onClick.When we click on element 2 then eventHandler for both the elements will be executed

  4. What is a callback function and why will you use it?

    5. Ans: A callback is a function that waits for another function to finish executing before executing itself. It gives us the ability to enforce an order of operations to our JavaScript code.
